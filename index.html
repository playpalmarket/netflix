Ketentuan & Bantuan Garansi

Masa Langganan

1 bulan = 25–30 hari (tergantung paket). Periode ini tetap dihitung sebagai “1 bulan” untuk garansi dan masa berlaku.

Ketentuan Garansi

  1. Dilarang mengubah informasi akun apa pun (email, password, dsb). Pelanggaran: denda Rp150.000.
  2. Login hanya melalui aplikasi/website resmi.
  3. Akun hanya untuk 1 perangkat pada saat yang sama.
  4. Dilarang login di perangkat lain tanpa logout dari perangkat sebelumnya.
  5. Jika pindah perangkat, logout dulu di perangkat lama.
  6. Masa garansi mengikuti durasi paket yang ditentukan.
  7. Pelanggaran ketentuan: garansi hangus dan denda sesuai aturan.

Netflix — Mengatasi PIN Profil Salah

  1. Pilih Lupa PIN / Forgot PIN di layar PIN profil.
  2. Masukkan password akun Netflix.
  3. PIN profil yang benar akan ditampilkan.
  4. Kembali, lalu masukkan PIN yang sudah dicek.

Jika “Lupa PIN” tidak muncul: pastikan aplikasi Netflix versi terbaru, coba via browser di netflix.com, atau hubungi admin untuk bantuan reset PIN.

Netflix — Tata Cara Kirim Kode Household

Lewat TV

  1. Pastikan profil yang terkena household berbahasa Indonesia atau English.
  2. Pastikan tertulis: “TV mu bukan bagian dari rumah dengan akun Netflix untuk akun ini”.
  3. Pilih Saya Sedang Bepergian.
  4. Pilih Kirim Email.
  5. Ambil foto layar (tangkapan layar) dan kirim ke owner.
  6. Owner mengirim kode, selesai.

Lewat HP

  1. Pastikan profil yang terkena household berbahasa Indonesia atau English.
  2. Pastikan tertulis: “TV mu bukan bagian dari rumah dengan akun Netflix untuk akun ini”.
  3. Pilih Tonton Sementara.
  4. Pilih Kirim Email.
  5. Ambil foto layar dan kirim ke owner.
  6. Owner mengirim kode, selesai.

Troubleshooting kirim gambar/URL

  • Jika muncul pesan “Gagal kirim gambar. Cek apakah URL-nya valid.” pastikan tautan dapat diakses publik dan tidak memerlukan login.
  • Gunakan layanan berbagi gambar dengan URL langsung (misalnya file statis GitHub Pages, Imgur, atau penyimpanan yang memberikan tautan file langsung).
  • Hindari URL yang berakhir pada halaman pratinjau; pastikan format file gambar (.jpg/.png/.webp) terbaca langsung.

SNK Netflix

  • Wajib clear data / reinstall app sebelum login (menghindari error/incorrect session).
  • Max login 1 perangkat. Jika ingin pindah device, wajib kabari admin terlebih dahulu.
  • Dilarang mengubah profil & PIN, billing/payment, email, dan password.
  • Wajib simpan SS bukti login; tanpa bukti, garansi tidak berlaku.

Penegakan & Denda

  • Login lebih dari 1 device → Denda Rp200.000.
  • Pelanggaran aturan (mengganti kredensial/otak-atik akun) → Denda Rp500.000 + Garansi hangus + Akun ditarik.
  • Akun dipantau; pola login tidak sesuai form dapat berakibat akun ditarik dan garansi hangus.
